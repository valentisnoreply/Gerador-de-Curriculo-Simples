# Curriculo Jovem Aprendiz

Gerador simples de curriculo em HTML/CSS/JS com formulario de edicao, previsualizacao em tempo real, layout tradicional A4 e exportacao via impressao (PDF).

## Recursos

- Edicao por formulario com campos preenchiveis.
- Mostrar/ocultar secoes via toggles.
- Layout moderno e layout tradicional A4.
- Paletas de cores alternaveis.
- Exportacao em PDF pelo dialogo do navegador.

## Como usar

1. Abra `index.html` no navegador.
2. Preencha os campos no painel de edicao.
3. Escolha layout e paleta de cores.
4. Clique em "Exportar PDF" e salve como PDF.

## Estrutura

- `index.html`: layout e formulario.
- `styles.css`: estilos e temas.
- `script.js`: logica de previsualizacao e toggles.
